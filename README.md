RemoteControlServer 1.0.0 Beta

This app runs a remote control server for Android devices that can be discovered by other devices on the same Wi-Fi network.
Features

* Full screen capture at 480p / 30 FPS (selectable app/window capture)
* Multi-touch input support using AccessibilityService
* Audio capture and streaming (internal audio, AAC)
* Raw H.264 video streaming over TCP
* Touch control, mouse, keyboard, and controller support
* Network discovery with QR code sharing
* No root access required
* Optional selective app capture for privacy and bandwidth efficiency
* Designed for gaming or general remote control
Extended Description

RemoteControlServer allows you to control an Android device remotely, with low latency if using a hotspot or local Wi-Fi network. You can pick a single app or mirror the full device screen. Input can be provided via touch, mouse, keyboard, or controllers.
This is a beta release (v1.0.0-beta) intended for testing. Some features may still have bugs. An iPadOS / iOS client is in development to connect to this server seamlessly.
How to Use

1. Install the app on your Android device.
2. Enable AccessibilityService for input handling.
3. Launch the app and start the server.
4. Use a compatible client on another device to discover and connect.
5. Select the app or screen to mirror (optional).
6. Control the device remotely with touch, keyboard, mouse, or controller.
Requirements

* Android 10+ for audio capture
* No root required
* Local network or hotspot for low-latency streaming
